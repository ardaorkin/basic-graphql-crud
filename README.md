## Backend

- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [GrahpQL](https://graphql.org/)

# Installation

## Frontend

To install packages run

```code
yarn
```

To start app run

```code
yarn dev
```

To build app run

```code
yarn build
```

## Backend

To install packages run

```code
yarn
```

To start server run

```code
yarn serve
```
